**Goal:** Extract SNI from QUIC Initial packets (UDP) and feed the existing domain/SNI pipeline. Vendored crypto (no new deps), reusing the existing ClientHello parser.

**Architecture:** New `quic` module in ppcap-core ÔÇö detect Initial Ôćĺ derive public initial keys (vendored HMAC/HKDF over the existing vendored SHA-256) Ôćĺ header unprotect + AES-128-GCM decrypt (vendored) Ôćĺ reassemble CRYPTO frames Ôćĺ reuse `fingerprint_tls_client_hello`/`sniff_tls_client_hello`. **Tech Stack:** Rust (ppcap-core, pure compute, wasm-safe). No new deps.

## Global Constraints

- **No new deps** in ppcap-core; vendor the crypto. C-free gate must stay empty; wasm-safe (`ppcap-wasm` still builds; no `std::{fs,net,time}` in `quic`).

- **Correctness is pinned by RFC vectors** ÔÇö crypto primitives verified against published KATs; the QUIC chain verified against RFC 9001 Appendix A's published intermediate values AND its protected packet. Decryption uses only public inputs.

```rust
use crate::analyze::sha256;

const BLOCK: usize = 64;

pub(crate) fn hmac_sha256(key: &[u8], msg: &[u8]) -> [u8; 32] {
    let mut k = [0u8; BLOCK];
    if key.len() > BLOCK {
        k[..32].copy_from_slice(&sha256(key));
    } else {
        k[..key.len()].copy_from_slice(key);
    }
    let mut ipad = [0x36u8; BLOCK];
    let mut opad = [0x5cu8; BLOCK];
    for i in 0..BLOCK {
        ipad[i] ^= k[i];
        opad[i] ^= k[i];
    }
    let mut inner = Vec::with_capacity(BLOCK + msg.len());
    inner.extend_from_slice(&ipad);
    inner.extend_from_slice(msg);
    let inner_hash = sha256(&inner);
    let mut outer = Vec::with_capacity(BLOCK + 32);
    outer.extend_from_slice(&opad);
    outer.extend_from_slice(&inner_hash);
    sha256(&outer)
}

pub(crate) fn hkdf_extract(salt: &[u8], ikm: &[u8]) -> [u8; 32] {
    hmac_sha256(salt, ikm)
}

pub(crate) fn hkdf_expand(prk: &[u8; 32], info: &[u8], out_len: usize) -> Vec<u8> {
    let mut out = Vec::with_capacity(out_len);
    let mut t: Vec<u8> = Vec::new();
    let mut counter: u8 = 1;
    while out.len() < out_len {
        let mut data = Vec::with_capacity(t.len() + info.len() + 1);
        data.extend_from_slice(&t);
        data.extend_from_slice(info);
        data.push(counter);
        t = hmac_sha256(prk, &data).to_vec();
        out.extend_from_slice(&t);
        counter = counter.wrapping_add(1);
    }
    out.truncate(out_len);
    out
}

/// TLS 1.3 HKDF-Expand-Label (RFC 8446 ┬ž7.1), empty context.
pub(crate) fn hkdf_expand_label(secret: &[u8; 32], label: &str, out_len: usize) -> Vec<u8> {
    let full = format!("tls13 {label}");
    let mut info = Vec::with_capacity(2 + 1 + full.len() + 1);
    info.extend_from_slice(&(out_len as u16).to_be_bytes());
    info.push(full.len() as u8);
    info.extend_from_slice(full.as_bytes());
    info.push(0u8); // empty context
    hkdf_expand(secret, &info, out_len)
}
```

- [ ] **Step 3: Implement AES-128 + GCM.** Use the **canonical** AES S-box and Rcon (FIPS-197 Figure 7 / ┬ž5.2) ÔÇö the C.1 test pins any transcription error. Structure:
  - `struct Aes128 { rk: [[u8;4];44] }` (11 round keys ├Ś 4 words). `new`: KeyExpansion (Rcon `01,02,04,08,10,20,40,80,1b,36`, SubWord via S-box, RotWord). `encrypt_block`: AddRoundKey Ôćĺ 9├Ś(SubBytes, ShiftRows, MixColumns, AddRoundKey) Ôćĺ final (SubBytes, ShiftRows, AddRoundKey). GF(2^8) `xtime` for MixColumns.
  - GCM:
    - `ghash(h: [u8;16], aad, ct) -> [u8;16]`: GF(2^128) (RFC 8452/SP 800-38D bit order ÔÇö MSB-first, reduction poly `0xe1<<120`). Blocks: each 16-byte block of AAD (zero-padded) then ciphertext (zero-padded), then a final block `u64be(aad_bits) ++ u64be(ct_bits)`; `X = (X ^ block) ┬Ě H`.
    - `H = Aes128::encrypt_block([0u8;16])`.
    - `J0 = nonce(12) ++ 0x00000001`. Keystream/tag: `tag = ghash(H, aad, ct) ^ Aes128::encrypt_block(J0)`. Plaintext: AES-CTR starting at `inc32(J0)` over the ciphertext.
    - `aes128_gcm_open`: split `ct_and_tag` into `ct` + 16-byte `tag`; recompute the expected tag; if `!= tag` Ôćĺ `None`; else decrypt CTR Ôćĺ `Some(plaintext)`.
```rust
pub(crate) fn aes128_gcm_open(key: &[u8;16], nonce: &[u8;12], aad: &[u8], ct_and_tag: &[u8]) -> Option<Vec<u8>> {
    if ct_and_tag.len() < 16 { return None; }
    let (ct, tag) = ct_and_tag.split_at(ct_and_tag.len() - 16);
    let aes = Aes128::new(key);
    let h = aes.encrypt_block(&[0u8;16]);
    let mut j0 = [0u8;16];
    j0[..12].copy_from_slice(nonce);
    j0[15] = 1;
    let s = ghash(&h, aad, ct);
    let ej0 = aes.encrypt_block(&j0);
    let mut expected = [0u8;16];
    for i in 0..16 { expected[i] = s[i] ^ ej0[i]; }
    if expected != tag[..16] { return None; }   // tag MUST verify
    // CTR decrypt from inc32(J0)
    let mut counter = j0; incr32(&mut counter);
    let mut out = Vec::with_capacity(ct.len());
    for chunk in ct.chunks(16) {
        let ks = aes.encrypt_block(&counter);
        for (i, &b) in chunk.iter().enumerate() { out.push(b ^ ks[i]); }
        incr32(&mut counter);
    }
    Some(out)
}
```
> NOTE: get GHASH's bit order right (the GCM `┬Ě` is over GF(2^128) with the reversed bit convention; SP 800-38D). The NIST KAT will fail loudly if the reflection/reduction is wrong ÔÇö fix until it passes. `incr32` increments the low 32 bits (big-endian) only.
